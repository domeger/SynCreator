# SynCreator
```
# clone the repository
$ git clone https://github.com/rcastley/SynCreator
$ cd SynCreator
$ python3 -m venv venv
$ . venv/bin/activate
$ pip3 install -e .
$ export FLASK_APP=flaskr
$ export FLASK_ENV=development
$ flask init-db
$ flask run -h 0.0.0.0 -p 8080
```
Open http://localhost:8080 in a browser.
